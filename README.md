This Student Information System is a full-stack application that connects a Java GUI to a MySQL database using JDBC. The core logic relies on a centralized connection class and "PreparedStatements" to securely handle data, which prevents common hacking techniques like SQL Injection. By automating the ID generation with MySQL's Auto-Increment feature, the app manages unique records without manual input, while a "Find" function allows the system to fetch and pre-fill existing data for easy editing.

The dashboard uses a JTable that acts as a live window into the database, utilizing a custom "update" method to refresh the view whenever data changes. To make the interface more professional and readable, I implemented a custom renderer that applies alternating row colors and adjusted row heights. This design ensures that the data is not only stored reliably in the backend but is also presented in a clean, user-friendly format in the frontend.
